This repository replicates, explains, and extends the paper on optimal stopping in sales conversations by Manzoor, Ascarza, and Netzer (2025).
The paper is available [here](https://www.hbs.edu/ris/Publication%20Files/26-029_64229d1d-cb43-48a6-b04c-891fce2b36bd.pdf) and the original GitHub repository can be accessed on [link](https://github.com/emaadmanzoor/stoppingagents/tree/main).
